git GIT "Comandos"
pwd : dice donde estamos ubicados

ls: muestra todos los archivos donde estamos ubicados 

ls -l para info para info detallada

cd : entrar a una carpeta con el nombre exacto 

cd .. : volver hacia atras

mkdir : crear carpetas

touch : para crear archivos

clear: limpiar la consola 

cd "nombre con espacio": para entrar a carpeta con espacio

git config --global user.name "Nombre" : Cuando trabajemos en equipo para saber quien se hecho el moco

git config --global user.email "email" 

git config --global --replace-all user.email "mail nuevo" ... es para remplazar si hiciste mal


Cada vez que quieramos crear versiones: 

comando que se aplica una sola vez dentro del proyecto. 

git init: para inicializar el repositorio en git. 

U: untraket - primer estado de git 

git status: en rojo signigica que estan en working directory primer estado
